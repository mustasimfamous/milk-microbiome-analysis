# milk-microbiome-analysis
#  (R Scripts)

This repository contains R scripts for six analyses of milk microbiome and milk traits.

## Scripts
- scripts_00_setup.R : Libraries and setup
- scripts_01_rare_taxa_milk.R : Rare taxa distribution in milk samples
- scripts_02_top20_dominant_paired_milk.R : Top 20 dominant species (17 paired cows)
- scripts_03_dominant_early_vs_late_mean.R : Dominant taxa (Early vs Late)
- scripts_04_milk_traits_paired.R : Milk traits comparison (Early vs Late)
- scripts_05_corr_heatmap_milk_traits.R : Correlation heatmap (taxa vs milk composition)
- scripts_06_alpha_beta_milk_early_late.R : Alpha & beta diversity (Early vs Late)
